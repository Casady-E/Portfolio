Portfolio

Bloom is a concept company/mobile application that I created as a personal project. Bloom is a delivery app for flowers and floral arrangements. The core feature of the app is linking the user with local floral shops and using a decentralized delivery service for optimal efficiency and speed. Due to such a broad user base, the brand identity for Bloom needed to be fun and inviting but still retain a professional aesthetic. After extensive competitor and market research I developed brand standards that would not ensure the brandmark stood out amongst its competitors. I used soft floral colors for the logo and included them in the wordmark as well to provide unity. I chose a rounded typeface for the wordmark so that it would compliment the style and aesthetic of the logo. I created this project using Illustrator.

Bloom App Showcase 1

Bloom is a concept company/mobile application that I created as a personal project. Bloom is a delivery app for flowers and floral arrangements. The Collections screen serves as the main hub to access specific items or curated collection. As seen in the interactive prototype, the different items are all swipeable with a simple gesture which keeps the interface clean for usability. As with most of the app design, I wanted to let the flowers really stand out since they are the sole reason that anyone would be using the app. I used rounded rectangles throughout the app design to provide an easy to follow design system and style guide for each layout. I incorporated bright and festive colors in the Collections page design so that the user would stay interested while scrolling through. On each Item page, there is the option to add the item to your cart as a gift. Since most flowers are sent as gifts, it is essential that this feature stands out and is easy to locate for the user. I used varying colors for each important element to allow the user to be drawn to different aspects of the page. For example, the review/rating star or the contrasting gift and Add To Cart buttons. A short description for each item provides a final selling point for the provider. I created this project using Photoshop, Illustrator, and InVision Studio.

Bloom App Showcase 2

Bloom is a concept company/mobile application that I created as a personal project. Bloom is a delivery app for flowers and floral arrangements. For the Shopping Cart screen, I made it easily maneuverable with all of the buttons having clear contrasting colors for fast usability. Since many orders would be in bulk, especially for weddings, I created a system to quickly add and subtract items from the cart. This is represented in the Interactive Prototype. The Shopping Cart screen is accessible on every page of the app so that a user can finalize their purchase at any point. The Shopping Cart icon turns into the green color when the user is on that specific page so that they know which screen they are on. The Menu screen is also accessible from any part of the app and has several useful links for the user. The user can simply click on the grayed out part of the Menu screen to return to the page that they were viewing. This allows for the user to navigate around the app with ease and find any specific item. I created this project using Photoshop, Illustrator, and InVision Studio.
