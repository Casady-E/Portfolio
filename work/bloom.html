Bloom App & Brand Design

Client

Personal Project

Role

Product Designer

Software

Photoshop // Illustrator // InVision Studio

Link to Interactive Prototype

Bloom is a concept company/mobile application that I created as a personal project. Bloom is a delivery app for flowers and floral arrangements. The core feature of the app is linking the user with local floral shops by using a decentralized delivery service for efficiency and speed. As seen in the interactive prototype, the different items are all swipeable with a simple gesture which keeps the interface clean for usability.
